GmapSearch is a Chrome extension designed to help you quickly get directions to places on Google Maps. With GmapSearch you can easily highlight any text on a web page, right-click, and chose "Search in Google Maps" from the context menu, which will then open a Google Maps window with search results you can use to get directions. No more manually copying and pasting place names. :rocket:

## Features
- Quickly search any place on Google Maps
- Easily get directions with a single click
- No more copying and pasting individual place names
- Supports both English and non-English characters

## Installation :wrench:
To install the extension, simply follow these steps: 
- Go to `chrome://extensions/`
- Enable Developer Mode (toggle in the upper right)
- Click “Load Unpacked”
- Select the root folder of the GmapSearch Chrome extension

## Usage
1. Highlight any text on any web page
2. Right-click and select "Search in Google Maps"
3. A Google Maps window will open with search results
4. Use the Google Maps results to get directions
